Package Management
When using Expo, it is highly recommended to use expo install instead of bun add for React Native libraries. This ensures you get the version compatible with your SDK.

Install a Library: bunx expo install <package-name>
Install Multiple: bunx expo install expo-router react-native-safe-area-context
Fix Dependencies: bunx expo install --fix

======== yaml =======

name: Deploy
on:
  push:
    branches: [main]

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: oven-sh/setup-bun@v1
      - run: bun install
      - uses: expo/expo-github-action@v8
        with:
          expo-version: latest
          eas-version: latest
          token: ${{ secrets.EXPO_TOKEN }} # Set in GitHub Secrets

      - name: Build and Submit
        run: bunx eas build --platform all --profile production --non-interactive --auto-submit
        env:
          EXPO_PUBLIC_API_URL: ${{ secrets.EXPO_PUBLIC_API_URL }}
          EXPO_PUBLIC_PROJECT_ID: ${{ secrets.EXPO_PUBLIC_PROJECT_ID }}
